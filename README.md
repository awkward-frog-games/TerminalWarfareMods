# Terminal Warfare Mods

These are example mods (games, terminal customizations) for the game [Terminal Warfare](https://store.steampowered.com/app/4812060/Terminal_Warfare/).

## Installation

You can copy the each folder from  mods/ into your Terminal Warfare save-directory mods path:

*  Linux:   ~/.local/share/love/terminal_warfare/mods/
*  macOS:   ~/Library/Application Support/LOVE/terminal_warfare/mods/
*  Windows: %APPDATA%/LOVE/terminal_warfare/mods/
