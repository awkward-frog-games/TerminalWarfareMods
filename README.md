# Terminal Warfare Mods

These are example mods (games, terminal customizations) for the game [Terminal Warfare](https://store.steampowered.com/app/4812060/Terminal_Warfare/).

## Mini-Game Mods

Most of these games are very minimal re-implementations of classic games, I don't own the copyright to any of these.

* choppercmd - A Chopper Commando clone
* dataworm   - A Snake clone
* interceptcmd - A Missile Command clone
* pipedream - A Pipe Dream style clone
* terrainbarrage - A Scorched Earth clone

## Customization Mods

These mods require added support from the core game, but I didn't want to include them in the codebase.

* ohmyzsh-prompts - A terminal prompt customization mod
* pokemon-colorscripts - Add Pokemon avatars to your terminal

## Installation

You can copy the each folder from  mods/ into your Terminal Warfare save-directory mods path:

*  Linux:   ~/.local/share/love/terminal_warfare/mods/
*  macOS:   ~/Library/Application Support/LOVE/terminal_warfare/mods/
*  Windows: %APPDATA%/terminal_warfare/mods/

If you are still on the demo, the mods should still load fine but the directory is slightly different:

*  Linux:   ~/.local/share/love/terminal_warfare_demo/mods/
*  macOS:   ~/Library/Application Support/LOVE/terminal_warfare_demo/mods/
*  Windows: %APPDATA%/terminal_warfare_demo/mods/
